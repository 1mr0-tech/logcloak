Features

Everything you need, nothing you don't

Works on k3s, GKE, EKS, and AKS with a single Helm install.

🛡️

Zero-config TLS

Self-signed certificate generated automatically on first install. CA bundle patched into the webhook configuration with no manual setup.

⚡

No latency impact

Apps write to a kernel FIFO — identical to any file descriptor write. Masking is asynchronous. The app never blocks waiting for masking.

🎯

Two-tier rule system

MaskingPolicy CRD for platform-wide baselines. Pod annotations for per-service extensions. Annotations always extend, never override.

🔐

Fail-closed

If a line cannot be processed, it is dropped and a [LOGCLOAK-DROP] sentinel is written. Raw PII never leaks on masker failure.

🌍

RE2-safe patterns

Only RE2-compatible regex allowed — linear time, immune to ReDoS. Unsafe patterns are rejected at pod admission before the app starts.

📊

Prometheus metrics

Built-in counters for lines processed, masked, and dropped, plus per-line latency histograms. Scrape on :9090/metrics.
