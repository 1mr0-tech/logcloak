logcloak

Helm Chart Repository

Real-time Kubernetes pod log masker. Redacts PII, OTPs, and tokens before logs reach kubectl logs or any storage backend.

Install

helm repo add logcloak https://1mr0-tech.github.io/logcloak
helm repo update
helm install logcloak logcloak/logcloak \
  --namespace logcloak \
  --create-namespace

Opt in a namespace

kubectl label namespace <your-namespace> logcloak.io/inject=true

Add rules via annotation

metadata:
  annotations:
    logcloak.io/patterns: "email,otp-6digit,jwt"
    logcloak.io/regex-order-id: "ORD-[0-9]{8}"
